# Company Profile

Website Company Profile Menggunakan Laravel 8

## Installation

You can fork or clone this project

```
git clone https://github.com/Fadhila36/company-profil.git
cd company-profil
composer install
cp .env.example .env <-- edit db config
php artisan admin:install
Php artisan serve
```
